Projects

Validate Manual

Validate Manual function crawls though the entire publication and checks for known pitfalls such as missing revision dates, graphic sizing errors, broken links, xml discrepancies, and change markup errors. The report is generated for the Technical Writer in less than 1 minute, and can save hours of work.

How it works--Validate Manual starts at the top of the manual and works its way through every single chapter, section, appendix collecting all data about data content, tables, graphics, links, dates, change controls. If there are errors within the collected data, a report is generated to help the Technical Writer correct the mistakes before the output is generated.
